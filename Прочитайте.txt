Программа для составления оперативного графика
Версия 1.0   16.03.2016г.


Данная программа позволяет скомпоновать оперативный план для технического обслуживания Ж.Д. устройств в файл формата MS Word 
В папке «Оперативный план» 4 папки (ГАЦ, Замедлители, АРС ГТСС, СЦБ), каждая служит для составления плана соответствующих типов устройств 

Замечания по установке:
- Для среды Windows сначала установите файл «jre» (среда выполнения JAVA - JRE), находящийся в корневой папке
- Данный файл для вашей ОС можно скачать по ссылке http://www.oracle.com/technetwork/java/javase/downloads

Инструкции по управлению программой:
- Сравнените, проверьте или введите данные во всех вкладках программы 
- На вкладке «Четырех-ный график» выберите месяц для составления оперативного плана и нажмите кнопку «Сформировать план»
- На вкладке «Оперативный график» вручную введите пункты годового графика в соответствующий столбец на форме
- При наведении курсора на пункт годового графика на правой панели под курсором появится соответственно название пункта
- Нажмите кнопку «Составить оперативный план» отчет готов!

ВАЖНО:
- Не забывайте в конце каждого года вносить изменения и проверять календарь (сохраняется в файл Calendar.out). От этого зависит правильность составления графика
- Не забывайте вручную проставлять пункты годового графика на вкладке «Оперативный график»
